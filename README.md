## How To Use
> Follow the steps below
```shell

# Clone the repository
git clone https://github.com/nandajfa/cub3D.git

# Access the folder
cd cub3D

# Run make to compile the program
make

# Run the program
./cub3d <.cub file>

# Clean output files with
make fclean

# Finish
```
